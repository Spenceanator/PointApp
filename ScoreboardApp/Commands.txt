CMD Commands
CD Change directory
CD.. goes up one directory
G: change drive
DIR display everything in directory
NUL>blah or blah.txt || Creates file
del /f filename delete file
rmdir to remove directory
Press F7 to display commands used
filename to open file || example: commands.txt

git status || Show what's different between us and what's currently uploaded
git commit -m "Message" ||Commits but isn't synced up with github.com
git add . |&&| git add -A || Will add everything that is in the directory
